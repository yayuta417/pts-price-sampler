# PTS Price Sampler

個人利用目的の株価サンプリング・ランナーです。日本のPTS（私設取引システム）に取次がれた銘柄の公表済み最新値を、取引セッション中に毎分1回取得し、プライベートのデータリポジトリへ記録します。

- 対象銘柄・保存先リポジトリは Actions Secrets（`STOCK_CODE` / `DATA_REPO` / `DATA_REPO_DEPLOY_KEY`）で指定するため、このリポジトリにはデータを置きません。
- アクセス頻度は毎分1リクエスト（セッション時間帯のみ）の低負荷運用です。
- スケジュール: 寄り前 08:18〜09:02 JST、夜間 16:25〜00:01 JST（平日）。

## 構成

| ファイル | 役割 |
|---|---|
| `PTS収集ポーラー.ps1` | サンプリング本体（pwsh、毎分取得→CSV追記→定期push） |
| `.github/workflows/PTS夜間収集.yml` | 夜間セッション（6時間のジョブ上限対策で3ジョブ直列） |
| `.github/workflows/PTS寄り前収集.yml` | 寄り付き前（8:20〜9:00）の収集 |
